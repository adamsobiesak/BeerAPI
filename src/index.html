The Beer Finder App

We know everything about beers!

I feel like searching for a beer by...

  • NAME
  • DESCRIPTION
  • BITERNESS (IBU)
  • ALCOHOL VOLUME (ABV)

This app was created as CodersCamp 2019 project by a team from Warsaw Group. We have chosen using Punk API to create an app for beer lovers.
